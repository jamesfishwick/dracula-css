Web Components

Dracula CSS provides ::part() selectors for styling web component internals. Component authors expose parts, and these styles apply automatically.

Common Part Names

The following ::part() selectors are styled:

  • ::part(button), ::part(button-primary), ::part(button-secondary)
  • ::part(input), ::part(textfield)
  • ::part(label), ::part(helper), ::part(error)
  • ::part(panel), ::part(container)
  • ::part(header), ::part(content), ::part(footer)
  • ::part(badge), ::part(chip), ::part(tag)
  • ::part(listitem), ::part(menuitem)
  • ::part(overlay), ::part(backdrop)

Demo: Custom Element with Parts

Usage in Your Components

// Your component template:
// <template>
//   <div part="panel">
//     <h3 part="header">Title</h3>
//     <div part="content"><slot></slot></div>
//     <button part="button">Action</button>
//   </div>
// </template>

// Dracula CSS styles these parts automatically!

Syntax Highlighting Integration

Dracula CSS provides container styling for Prism.js, Highlight.js, and Shiki. Add the official Dracula theme for token colors.

Code Block with Header

dracula.config.js JavaScript 
const dracula = {
  colors: {
    background: '#282a36',
    foreground: '#f8f8f2',
    purple: '#bd93f9',
    pink: '#ff79c6',
    cyan: '#8be9fd',
    green: '#50fa7b'
  },

  apply(element) {
    element.style.background = this.colors.background;
    element.style.color = this.colors.foreground;
  }
};

export default dracula;
